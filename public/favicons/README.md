# Favicons

Placez vos fichiers favicon ici :

- `favicon-light.png` - Favicon pour le mode clair (fond clair)
- `favicon-dark.png` - Favicon pour le mode sombre (fond sombre)
- `apple-touch-icon.png` - Icône pour iOS (180x180px recommandé)

Les fichiers seront automatiquement référencés dans le Layout.
